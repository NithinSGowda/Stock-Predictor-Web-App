API

Documentation for Stock predictor API

API basics

- Available on GET requests
- Prediction Data
- Endpoint : https://api.stockpredict.ml/data/company_name
- Previous data (Advanced)
- Endpoint : https://api.stockpredict.ml/data/full/company_name
Note : Requests are cached for 24 hours
